# Tutorial detallado de Git (ruta completa)

Este tutorial está pensado para estudiantes que comienzan desde cero.

## Objetivo

Aprender Git y colaboración profesional con flujo real:

- ramas
- commits claros
- pull requests
- trabajo en equipo con fork
- resolución de conflictos

## Estructura del curso Git en esta carpeta

- `modulo-0-onboarding/`
- `modulo-1-basico/`
- `modulo-2-commits/`
- `modulo-3-ramas/`
- `modulo-4-pr/`
- `modulo-5-colaboracion/`
- `modulo-6-conflictos/`
- `proyecto-final/`

Cada módulo incluye:

- `README.md` (explicación)
- `ejercicio.md` (práctica)
- `solucion.md` (referencia)

## Comandos base que debes dominar

```bash
git status
git switch -c feature/nombre
git add .
git commit -m "mensaje claro"
git push origin feature/nombre
```
