# Modulo 02: Procesamiento de datos

## Objetivo

Aprender el flujo basico de transformacion de datos: leer, limpiar, transformar y escribir resultados.

## Conceptos clave

- ETL y ELT
- calidad de datos (nulos, tipos, duplicados)
- trazabilidad del procesamiento

## Practica guiada

1. Tomar un archivo desde `data/lake/raw`.
2. Limpiar columnas con valores nulos o inconsistentes.
3. Generar un resultado agregado simple.
4. Guardar salida en `data/lake/processed`.

## Entregable

- script de procesamiento
- salida procesada en carpeta `processed`
- descripcion corta de reglas aplicadas
