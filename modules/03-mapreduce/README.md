# Modulo 03: MapReduce (mentalidad distribuida)

## Objetivo

Comprender el modelo MapReduce como base conceptual del procesamiento distribuido.

## Conceptos clave

- fase `map`: transformar registros en pares clave-valor
- fase `reduce`: agrupar y consolidar resultados
- escalabilidad horizontal

## Practica guiada

1. Implementar un conteo de palabras simple.
2. Definir claramente clave y valor en fase `map`.
3. Consolidar conteos en fase `reduce`.

Archivo sugerido:

- `spark/jobs/wordcount.py` (o equivalente)

## Entregable

- script funcional de conteo
- breve explicacion de `map` y `reduce` usados
