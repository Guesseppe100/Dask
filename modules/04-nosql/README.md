# Modulo 04: NoSQL con MongoDB

## Objetivo

Entender cuando usar NoSQL y realizar operaciones basicas en MongoDB para analitica.

## Conceptos clave

- modelo documental
- diferencia entre SQL y NoSQL
- diseno de documentos para consulta rapida

## Practica guiada

1. Levantar perfil `nosql`:

```bash
docker compose --profile nosql up -d
```

2. Insertar registros de ejemplo.
3. Consultar por filtros y agregaciones simples.

## Entregable

- coleccion creada con datos del ejercicio
- 3 consultas documentadas (find, filtro, agregacion)
