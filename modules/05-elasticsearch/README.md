# Modulo 05: Busqueda y analitica con Elasticsearch

## Objetivo

Aprender busqueda de texto y consultas analiticas basicas sobre indices.

## Conceptos clave

- indice
- documento
- consulta `match` y filtros
- visualizacion en Kibana

## Practica guiada

1. Levantar perfil `search`:

```bash
docker compose --profile search up -d
```

2. Indexar dataset simple.
3. Ejecutar consultas de texto.
4. Explorar resultados en Kibana.

## Entregable

- indice creado
- ejemplos de consultas
- captura o descripcion del dashboard en Kibana
