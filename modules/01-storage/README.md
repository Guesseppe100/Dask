# Modulo 01: Storage y Data Lake basico

## Objetivo

Entender donde guardar datos en proyectos de Big Data y como organizar carpetas de manera profesional.

## Conceptos clave

- diferencia entre datos crudos (`raw`) y procesados (`curated`)
- estructura de data lake por capas
- convenciones de nombres y particiones

## Practica guiada

1. Revisar carpeta `data/`.
2. Crear estructura:
   - `data/lake/raw`
   - `data/lake/processed`
3. Copiar un CSV de ejemplo a `raw`.
4. Documentar metadatos minimos del archivo (origen, fecha, columnas).

## Entregable

- estructura de carpetas creada
- archivo cargado en `raw`
- nota breve con metadatos del dataset
